# Customer Churn Analytics & Retention Intelligence System

## Overview
End-to-end data analytics project to predict customer churn and optimize retention strategies.

## Tech Stack
- Python (Pandas, ML)
- SQL
- Power BI

## Features
- Churn prediction model
- Customer segmentation
- Revenue loss analysis
- Interactive dashboard

## Key Insights
- Month-to-month contracts have highest churn
- New customers churn more
- High charges increase churn risk
